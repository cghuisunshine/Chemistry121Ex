1) Vector vs Wave Function

User

vector vs wave function

Assistant

Big picture: A vector is an abstract object in a vector space; a wave function is a coordinate representation of a quantum state vector in position space.

State vector (abstract):

\(\;|\psi\rangle\;\) lives in an abstract Hilbert space (basis-independent).

Wave function (representation):

\(\;\psi(x)=\langle x|\psi\rangle\;\) is the position-space representation; \(|\psi(x)|^2\) is a probability density.

Core relationship:

\(\;\psi(x)=\langle x|\psi\rangle\;\) and (for a complete basis) \(\;|\psi\rangle=\int \psi(x)|x\rangle\,dx.\)

2) Same state in different “spaces” (bases)

User

it means the same state represent in different space?

Assistant

Yes—but “different space” here means different bases / coordinate representations within the same abstract Hilbert space.

The state itself is \(|\psi\rangle\). Choosing different complete bases gives different coordinate descriptions:

  • Position space: \(\psi(x)=\langle x|\psi\rangle\)
  • Momentum space: \(\phi(p)=\langle p|\psi\rangle\)
  • Energy basis: \(c_n=\langle n|\psi\rangle\)

Analogy: Same vector \(\vec v\) can be written in Cartesian vs spherical coordinates—the numbers change, the vector doesn’t.

User

and these space are all fully represent the same vector?

Assistant

Yes, as long as the basis is complete. Completeness is expressed by a resolution of the identity:

  • Position: \(\displaystyle \int |x\rangle\langle x|\,dx = \mathbf{1}\)
  • Momentum: \(\displaystyle \int |p\rangle\langle p|\,dp = \mathbf{1}\)
  • Discrete energy: \(\displaystyle \sum_n |n\rangle\langle n| = \mathbf{1}\)

Then the same state vector can be reconstructed in any complete basis:

[ |\psi\rangle = \int \psi(x)\,|x\rangle\,dx = \int \phi(p)\,|p\rangle\,dp = \sum_n c_n\,|n\rangle. ]

Not fully representative examples: incomplete bases, truncations, or keeping only partial information (e.g., only one component).

3) What is \(|x\rangle\langle x|\)?

User

what is this operator?

Assistant

\(\boxed{|x\rangle\langle x|}\) is a projection operator (an outer product) onto the position eigenstate \(|x\rangle\).

Type-check:

  • \(|x\rangle\) is a ket (vector)
  • \(\langle x|\) is a bra (dual vector)
  • \(|x\rangle\langle x|\) is an operator

Action on a state:

[ |x\rangle\langle x|\psi\rangle = |x\rangle\,\psi(x) \quad\text{because}\quad \langle x|\psi\rangle=\psi(x). ]

Measurement connection:

[ \langle \psi|\,|x\rangle\langle x|\,|\psi\rangle = |\psi(x)|^2. ]

Note: \(|x\rangle\) is a generalized eigenket (not normalizable in the usual sense), so these objects are handled with distributions (Dirac delta) and integrals.

4) Can you swap \(\psi(x)\) and \(|x\rangle\)?

User

could I exchange the position as ψ(x)x⟩

Assistant

Yes. Since \(\psi(x)\) is a complex scalar, it commutes with kets:

[ |x\rangle\,\psi(x) \;=\; \psi(x)\,|x\rangle. ]

Why: \(\psi(x)=\langle x|\psi\rangle\) is a number. Scalars can be moved left/right without changing meaning.

Important contrast: you generally cannot reorder operators freely, e.g. \(\hat X|\psi\rangle \neq |\psi\rangle\hat X\) (the right-hand expression isn’t even well-defined in standard notation).

5) Proof: \(\int |x\rangle\langle x|dx = \mathbf{1}\)

User

prove this

Assistant

We prove the operator identity

[ \boxed{\int_{-\infty}^{\infty} |x\rangle\langle x|\,dx \;=\; \mathbf{1}} ]

by showing both sides act the same on an arbitrary state \(|\psi\rangle\).

Step 1 — Apply LHS to \(|\psi\rangle\):

[ \left(\int |x\rangle\langle x|\,dx\right)|\psi\rangle = \int |x\rangle\langle x|\psi\rangle\,dx. ]

Step 2 — Use \(\langle x|\psi\rangle=\psi(x)\):

[ \int |x\rangle\langle x|\psi\rangle\,dx = \int |x\rangle\,\psi(x)\,dx. ]

Define \(|\phi\rangle \equiv \int |x\rangle\,\psi(x)\,dx\). To show \(|\phi\rangle=|\psi\rangle\), compare their components in the position basis.

Step 3 — Test with \(\langle y|\):

[ \langle y|\phi\rangle = \left\langle y\left|\int |x\rangle\,\psi(x)\,dx\right.\right\rangle = \int \langle y|x\rangle\,\psi(x)\,dx. ]

Step 4 — Use \(\langle y|x\rangle=\delta(y-x)\):

[ \langle y|\phi\rangle = \int \delta(y-x)\,\psi(x)\,dx = \psi(y). ]

But \(\psi(y)=\langle y|\psi\rangle\). So \(\langle y|\phi\rangle=\langle y|\psi\rangle\) for all \(y\), implying \(|\phi\rangle=|\psi\rangle\).

Conclusion:

[ \left(\int |x\rangle\langle x|\,dx\right)|\psi\rangle = |\psi\rangle \quad\text{for all }|\psi\rangle \;\Rightarrow\; \int |x\rangle\langle x|\,dx = \mathbf{1}. ]

Assumptions used: completeness of \(\{|x\rangle\}\), normalization \(\langle x|x'\rangle=\delta(x-x')\), and Dirac’s generalized eigenvector framework (rigged Hilbert space in rigorous treatments).
